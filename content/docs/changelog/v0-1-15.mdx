### Features

- Redesigned play page with improved layout and configuration flow

- Manual fishing hints — agent AI-powered card recommendations with confidence levels
- Autoclaim and autorepair options now available in manual fishing mode
- Redesigned user profiles with stats charts, time range selector, and category breakdowns
- Conquest crown leaderboard and user faction highlighting

- Mobile navigation sidebar and home page redesign
- Login via Discord
- Free access to games with Scroll of Fire (bypasses spark cost)
- PWA install quest with spark rewards
- Pet feed popover replaced with in-card overlay
- Action token support for faster manual gameplay
