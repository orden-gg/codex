A reference for terms and concepts you'll encounter across Fireball and Gigaverse.

## A

**Abstract Wallet** — The wallet system used by Gigaverse and Fireball. Created when you sign up for Gigaverse. Fireball uses the same wallet for authentication.

**Agent Mode** — Automated fishing mode where Fireball's agent AI plays the entire session for you.

**Autoclaim** — An option when starting a run that automatically claims energy from your ROMs before the job begins.

**Autorepair** — An option when starting a run that automatically repairs broken gear before the job begins.

## C

**Conquest** — The territory control game mode where factions compete for hexagonal tiles on a map.

## D

**Daily Limit** — The maximum number of runs you can do per day for each game type. Resets at the daily reset time.

**Daily Reset** — The time each day when dungeon and fishing daily limits refresh. Shown as a countdown in the navigation.

## E

**Energy** — The resource consumed to start dungeon and fishing runs. Regenerates over time and can be supplemented by ROM production.

## F

**Faction** — One of the groups in Gigaverse (Crusader, Overseer, Athena, Archon, Foxglove, Summoner, Chobo, Gigus). Affects Conquest, hatching, and some recipes.

## G

**Giga-Juice** — An effect in Gigaverse required to access Fireball's full features. Has an expiration date shown in the sidebar.

## H

**Hints** — Agent AI-powered suggestions shown during manual fishing. Display a recommended action, reasoning, and confidence level.

## J

**Job** — A queued task on Fireball's servers representing a batch of dungeon runs or fishing sessions. Tracked on the Jobs page.

**Juiced Mode** — An option for some game types that increases the daily run limit at the cost of higher energy consumption (3x for dungeons).

## L

**Lure** — Secondary fishing equipment. Up to 2 lures can be equipped per fishing session.

## M

**Manual Mode** — A fishing mode where you play each card yourself, with optional agent AI hints.

## P

**PWA** — Progressive Web App. Fireball can be installed from your browser for a native app experience with push notifications.

## R

**Reveal** — The action of viewing accumulated service results. Services run silently until you reveal, which shows everything since your last check.

**Rod** — Primary fishing equipment. One rod is equipped per fishing session.

**ROM** — A digital asset in Gigaverse that produces energy, shards, dust, and GIGUS dust over time. Managed on the ROMs page.

**ROMs** — The dashboard section showing aggregate ROM statistics, energy production, and claim/convert actions.

## S

**Scroll of Fire** — A permanent scroll that grants free access to game runs and all four automated services.

**Scroll of Noob** — A scroll granting a 5-day free trial of dungeon and fishing runs.

**Sparks** — Fireball's credit system. Consumed by automated actions and purchased with ETH. Not a cryptocurrency token.

**Services** — Background automations (Fish Seller, ROM Claimer, Loot Collector, Hatchery) that run on a schedule.

## T

**Round** — A time interval in Conquest after which territory changes are applied and the map updates.

## U

**Untouchable Supply** — Fish Seller configuration that preserves a set number of fish per rarity, only selling the excess.
